游戏设置

### 洗均牌库：
游戏开始前，将行动牌混洗成一叠，放在桌子中央作为牌库。

### 确定阵营：
游戏开始前，玩家们随机抽取一张0层梦境牌，其中抽中背面为梦主标识牌的玩家担当梦主，其余玩家皆为盗梦者。

### 盗梦者设置：
1.每人随机派发3张盗梦者角色牌，各选取其中1张，背面朝上盖放于自己面前。
2.派发给每位盗梦者两种梦境牌各1张。梦境牌都有两面，一种是1和2、另一种是3和4，分别代表四层梦境。
3.盗梦者的梦境牌必须叠放在一起，放在最上面的数字代表该玩家现在身处的梦境层数，所有玩家游戏开始时均处于第1层梦境。

### 梦主设置：

1. 随机派发3张梦主角色牌，选取其中1张，背面朝上盖放于自己面前。
2. 将4张梦境牌按1、2、3、4的顺序排列于自己面前。
3. 将4张金库按自己的意愿，背面朝上盖放在四层梦境对应的位置，4张金库里面有1张是秘密，不能被盗梦者找到，其他3张是金币。
4. 按照配置表设置心锁的数量，用心锁骰子表示。将梦主指示物放置于第1层梦境牌上。
5. 梦主拥有1张特殊牌——梦主优势/M4卡宾枪（详见梦主的四大优势）。
6. 按照配置表选出本场游戏所需的贿赂牌，将其混洗并背面朝上放于一旁。游戏过程中，任何玩家包括梦主都不得查看这叠贿赂牌。
7. 梦主随机派发4张梦魇牌，查看并选择在每层梦境的金库下盖放一张梦魇牌。（详见梦魇牌）

例，6人游戏时，梦主的设置如下：
M4卡宾枪、梦主牌、贿赂牌、梦魇牌。
梦主指示物：1 2 3 4 层梦境共四张指示牌，其中一层有一个梦主指示物（国际象棋的兵），表示梦主所在层数。
心锁骰子：4张金库牌，分别对应4层梦境，上边各有一个骰子，点数表示还需要解锁的次数(具体次数按玩家人数指定)

游戏没有起始手牌

游戏提示

让玩家可以随时查看游戏规则。盗梦十诫是《盗梦都市》游戏过程中十条基本规则的提示卡。配置表则是心锁与贿赂牌的数据对照。

初次游戏时，可以考虑不加入角色牌和【移形换影】，即标有"EX"的行动牌。这样会令游戏更容易上手，过程更为流畅。

一局之后，玩家们已经对《盗梦都市》的规则有一定了解，那么就请马上加入角色牌和EX行动牌，体验完整的游戏过程吧。
